features:
  - icon: 🚀
    title: One-click start for multiple services
    details: Unified management of multiple application services, supporting batch start and stop, bid farewell to the tedious operation of manual execution one by one, making deployment a breeze.

  - icon: 🔄
    title: Auto-restart
    details: In case of abnormal service exit, automatic detection and restart will ensure the continuous and stable operation of the application, eliminating the need for manual monitoring.

  - icon: ⚡
    title: Start with the system
    details: It supports automatic startup upon Windows booting, ensuring the service is always online, eliminating the need to manually start it each time.

  - icon: 🖥️
    title: Built-in terminal console
    details: Each service independently possesses a terminal panel, which displays standard output and error logs in real-time. It supports ANSI color output, making debugging clear at a glance.

  - icon: 📊
    title: Real-time resource monitoring
    details: The main interface displays the real-time usage of CPU, memory, GPU, and video memory, allowing users to grasp the status of system resources and allocate service loads reasonably.

  - icon: 🔍
    title: Process and port tracing
    details: Automatically identify and display the PID and listening port number of each service, quickly locate the service running status, and troubleshoot issues more efficiently.

  - icon: 🛠️
    title: Flexible path configuration
    details: Supports both absolute and relative path configurations, adapts to different deployment scenarios, and eliminates the need to modify configurations individually during migration.

  - icon: 🌐
    title: Multilingual interface
    details: With multiple language packs built-in, users can switch between them at any time in the settings, catering to different usage habits.

  - icon: 🎨
    title: Terminal personalized customization
    details: Support for customizing terminal background colors, text colors, and font sizes allows for the creation of a personalized visual style, ensuring that prolonged use does not lead to fatigue.
