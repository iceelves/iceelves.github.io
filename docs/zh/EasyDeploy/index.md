features:
  - icon: 🚀
    title: 一键启动多服务
    details: 统一管理多个应用服务，支持批量启动与停止，告别逐个手动执行的繁琐操作，让部署变得轻而易举。

  - icon: 🔄
    title: 自动重启
    details: 服务异常退出时自动检测并重启，保障应用持续稳定运行，无需人工值守。

  - icon: ⚡
    title: 随系统自启
    details: 支持跟随 Windows 开机自动启动，服务始终在线，省去每次手动拉起的步骤。

  - icon: 🖥️
    title: 内置终端控制台
    details: 每个服务独立拥有终端面板，实时展示标准输出与错误日志，支持 ANSI 彩色输出，调试一目了然。

  - icon: 📊
    title: 实时资源监控
    details: 主界面实时显示 CPU、内存、GPU、显存占用情况，掌握系统资源状态，合理分配服务负载。

  - icon: 🔍
    title: 进程与端口追踪
    details: 自动识别并展示每个服务的 PID 与监听端口号，快速定位服务运行状态，排查问题更高效。

  - icon: 🛠️
    title: 灵活的路径配置
    details: 支持绝对路径与相对路径两种配置方式，适配不同部署场景，迁移环境无需逐一修改配置。

  - icon: 🌐
    title: 多语言界面
    details: 内置多种语言包，可在设置中随时切换，满足不同用户的使用习惯。

  - icon: 🎨
    title: 终端个性化定制
    details: 支持自定义终端背景色、文字颜色与字体大小，打造专属的视觉风格，长时间使用也不疲劳。
