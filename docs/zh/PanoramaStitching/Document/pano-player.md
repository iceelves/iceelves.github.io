# 全景投屏
全景视频通过多路或单路视频输入，经由全景拼接解码或使用本地解码，以实现低延迟拼接显示。系统依据拼接屏的物理布局及每块屏体的实际分辨率，对全景进行高精度区域裁切并完成多视窗映射。

![全景投屏](/PanoramaStitching/PanoPlayer.jpg)

| 播放方式 | 特点说明 |
| ---- | ---- |
| **低延时播放** | 实时获取相机码流拼接播放，延迟低； |
| **码流播放**   | 获取拼接软件推流的全景视频播放，延迟稍高； |
